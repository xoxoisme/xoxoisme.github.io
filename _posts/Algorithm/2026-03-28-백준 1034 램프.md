## 유형
> 브루트포스

## 풀이
- 열을 기준으로 해당 열의 모든 스위치를 반대로 변경한다.
- 핵심은 행이 만약 값이 같다면, 열을 뒤집을 시 모든 행에 동일하게 적용된다는 것이다.
- 또한, 남은 횟수가 짝수여야 횟수를 소모시킬 수 있다.(스위치는 두번 터치하면 그대로이기 때문에)

---

## 코드

```python
import sys

input = sys.stdin.readline

n, m = map(int, input().split())
lst = [input().rstrip() for _ in range(n)]
k = int(input())

max_cnt = 0

for row in range(n):
    zero = lst[row].count('0')

    same_rows = 0

    if zero <= k and (k - zero) % 2 == 0:
        for other_row in range(n):
            if lst[row] == lst[other_row]:
                same_rows += 1

    max_cnt = max(max_cnt, same_rows)

print(max_cnt)
```
