---

## 유형
> Greedy

## 풀이
- 역 사전순이기에 최대값이 앞자리로 와야한다.
- 현재 인덱스에서 **최대로 움직일 수 있는 횟수** 중 최대값을 찾는다.
- 여기서 최대로 움직일 수 있는 횟수는 `(현재 인덱스) + S + 1`이다.
- 최대 횟수가 배열의 최대 크기보다 크면 안 되도록 설정해준다.

---

## 코드

```python
N = int(input())
arr = list(map(int, input().split()))
S = int(input())

for i in range(N):
    if S <= 0:
        break

    max_idx = i
    for j in range(i + 1, min(N, i + S + 1)):
        if arr[j] > arr[max_idx]:
            max_idx = j

    for j in range(max_idx, i, -1):
        arr[j], arr[j - 1] = arr[j - 1], arr[j]

    S -= (max_idx - i)

print(*arr)
```
