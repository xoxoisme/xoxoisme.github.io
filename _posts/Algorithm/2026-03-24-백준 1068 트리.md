## 유형
> DFS
> 그래프 탐색

## 풀이
- 삭제할 노드는 의미없는 수로 삭제한다.
- 재귀를 돌며 해당 삭제할 노드와 삭제할 노드의 리프 노드까지 재귀로 의미없는 수를 넣어준다.
- 이후 의미없는 수와 트리 내에 리프가 없으면 카운트 한다.

### 다만, 전체를 탐색하는 배열 방식보다 자식 접근이 바로 가능한 그래프 방식으로도 풀어보았다.

---

## 코드

### DFS 방식

```python
import sys

n = int(sys.stdin.readline())
tree = list(map(int, sys.stdin.readline().split()))
delete = int(sys.stdin.readline())

def dfs(del_node):
    tree[del_node] = -2
    for i in range(n):
        if del_node == tree[i]:
            dfs(i)

dfs(delete)

res = 0
for i in range(n):
    if tree[i] != -2 and i not in tree:
        res += 1

print(res)
```

### 그래프 탐색 방식

```python
import sys

n = int(sys.stdin.readline())
tree = list(map(int, sys.stdin.readline().split()))
delete = int(sys.stdin.readline())

graph = [[] for _ in range(n)]
root = -1

for i in range(n):
    if tree[i] == -1:
        root = i
    else:
        graph[tree[i]].append(i)

deleted = [False] * n

def dfs(del_node):
    deleted[del_node] = True
    for child in graph[del_node]:
        dfs(child)

dfs(delete)

cnt = 0
for i in range(n):
    if deleted[i]:
        continue

    is_leaf = True
    for child in graph[i]:
        if not deleted[child]:
            is_leaf = False
            break

    if is_leaf:
        cnt += 1

print(cnt)
```
