## 유형
> 그리디

## 풀이
- **2진수**를 생각하여 풀면 쉽다.
- 원래 브루트포스로 1씩 더해가며 풀었는데, 시간을 줄일 수 있는 방법을 고민했다.
- 바로, 변환된 이진수를 역으로 돌려 **가장 낮은 자리의 1을 찾고**, 해당 위치 만큼 `2^idx`를 통해 한번에 올린다.
- 이러면, 전체 1의 개수를 구하는 연산을 확연히 줄일 수 있게 된다.

---

## 코드

### 브루트포스

```python
import sys

N, K = map(int, sys.stdin.readline().split())


count = 0
while bin(N).count('1') > K:
    N += 1
    count += 1

print(count)
```

### 그리디

```python
import sys

N, K = map(int, sys.stdin.readline().split())


count = 0
while bin(N).count('1') > K:
    idx = bin(N)[::-1].index('1')
    N += 2 ** idx
    count += 2 ** idx

print(count)
```
