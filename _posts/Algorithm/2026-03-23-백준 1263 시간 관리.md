## 유형
> 그리디 알고리즘

## 풀이
- 늦잠을 자는 것이 목적이기에 최대 마감기간에서부터 빼면서 진행한다.
- 뒤에서 부터 순서대로 (마감 기간) - (작업 시간) 을 진행하다가 마감 기간이 이른 작업이 나오면 해당 시간을 최대 마감기간으로 설정한다.
- 만약 0시부터 시작해도 작업을 해낼 수 없다면 -1을 반환한다.

---

## 코드

```python
import sys

N = int(input())
    
arr = []

for _ in range(N):
    a, b = map(int, sys.stdin.readline().split())
    arr.append((b, a))

arr.sort(reverse=True)
ans = arr[0][0] - arr[0][1]

for i in range(1, N):
    if ans > arr[i][0]:
        ans = arr[i][0] - arr[i][1]
    else:
        ans -= arr[i][1]
        
print(ans if ans >= 0 else -1)
```
