## 유형
> Stack

## 풀이
- 배열로도 가능하지만, **필요 없는 값에 대한 리소스가 낭비**된다.
- 스택을 이용하여 필요 없는 왼쪽 값을 `pop`을 통해 삭제한다.

---

## 코드

```python
import sys
input = sys.stdin.readline

N = int(input())
towers = list(map(int, input().split()))

stack = []
answer = [0] * N
for i in range(N):
    while stack:
        if stack[-1][1] > towers[i]:
            answer[i] = stack[-1][0] + 1
            break
        else:
            stack.pop()
    stack.append((i, towers[i]))
print(*answer)
```
