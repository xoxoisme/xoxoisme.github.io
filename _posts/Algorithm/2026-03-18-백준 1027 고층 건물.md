## 유형
> 브루트포스 알고리즘

## 풀이
- 두 건물의 선분 A와 B의 기울기를 구하는 문제이다.
- 배열로 각 건물이 볼 수 있는 건물의 개수를 저장한다.
- 왼쪽부터 오른쪽으로 이동하며 최대 기울기보다 큰 기울기가 나오면 1을 추가해준다.
- **A빌딩에서 B빌딩이 보인다면, B빌딩도 A빌딩이 보인다는 것**이다.
- 오른쪽으로만 이동하지만, 왼쪽으로 보이는 건물의 개수도 같이 구할 수 있다.

---

## 코드

```python
import sys

n = int(input())
heights = list(map(int, input().split()))
res = [0] * n

for i in range(n - 1):
    tmp = -sys.maxsize
    for j in range(i + 1, n):
        if tmp < (heights[j] - heights[i]) / (j - i):
            res[i] += 1
            res[j] += 1
            tmp = (heights[j] - heights[i]) / (j - i)

print(max(res))
```
