> "일단 끝까지 가보고, 막히면 돌아온다."

---

## 목표

> DFS(깊이 우선 탐색)에 대해서 알아봅니다.

---

## 기본 원리

- **한 방향으로 갈 수 있는 데 까지 깊이 내려간 다음**, 더 갈 곳이 없으면 이전 분기점으로 되돌아와서 다른 방향을 탐색한다.

---

## 코드

#### 재귀

```python
def dfs(node, visited):
    if node in visited:
        return
    visited.add(node)
    print(node)  # 방문 처리
    for neighbor in graph[node]:
        dfs(neighbor, visited)
```

- 재귀를 호출하여 시스템 스택에 쌓이면서 ‘**깊게 들어갔다 돌아오는**’ 동작이 일어난다.

```
[예시]

graph = {
    'A': ['B'],
    'B': ['D', 'C'],   # B의 이웃: D 먼저, 그다음 C
    'D': [],           # D는 막다른 길
    'C': []
}

--------------------------------------------------------------------------------------

[흐름]

dfs('A') 시작
│ print A
│ for neighbor in ['B']:
│   neighbor = 'B'
│   dfs('B') 호출 ──────────────┐
│                              │ dfs('B') 시작
│                              │ print B
│                              │ for neighbor in ['D', 'C']:
│                              │   1회차: neighbor = 'D'
│                              │   dfs('D') 호출 ──────────┐
│                              │                          │ dfs('D') 시작
│                              │                          │ print D
│                              │                          │ for neighbor in []:
│                              │                          │   (이웃 없음, 루프 안 돔)
│                              │                          │ dfs('D') 끝, 리턴
│                              │   ← 여기로 돌아옴 ────────┘
│                              │   2회차: neighbor = 'C'   ← 자동으로 다음 차례!
│                              │   dfs('C') 호출 ──────────┐
│                              │                          │ dfs('C') 시작
│                              │                          │ print C
│                              │                          │ (이웃 없음)
│                              │                          │ dfs('C') 끝, 리턴
│                              │   ← 여기로 돌아옴 ────────┘
│                              │ for 루프 끝
│                              │ dfs('B') 끝, 리턴
│   ← 여기로 돌아옴 ────────────┘
│ for 루프 끝
│ dfs('A') 끝
```

#### 스택

```python
def dfs(start):
    visited = set()
    stack = [start]
    while stack:
        node = stack.pop()  # LIFO
        if node in visited:
            continue
        visited.add(node)
        for neighbor in graph[node]:
            stack.append(neighbor)
```

- 사실 상 재귀도 스택을 이용하는 것이기에 같은 의미이다.
- 스택의 **LIFO(Last In First Out)** 을 활용한다. (↔BFS는 FIFO)

---

## 복잡도

> V = 정점(Vertex) 개수, E = 간선(Edge) 개수, h = 그래프의 깊이, w = 그래프의 최대 너비

- 시간 복잡도: **O(V+E)**
    - 모든 정점을 한 번씩 방문하고, 각 정점에서 인접한 모든 간선을 한번씩 확인한다.
- 공간 복잡도: **O(h)**
    - DFS는 한 쪽으로 끝까지 파고 들기에 현재 높이의 노드들을 스택에 들고 있다.
    - 다만, Visited도 따로 필요하기에 O(h+V)라고도 한다.
