> "하나의 출발점에서 다른 모든 정점까지의 최단 경로를 찾는다."

---

## 목표

> Dijkstra(다익스트라)에 대해서 알아봅니다.

---

## 기본 원리

- 다이나믹 프로그래밍(DP-기억하며 풀기)을 활용한 대표적인 최단 경로 탐색 알고리즘이다.
- 쉽게 말해, 현재까지 알고 있던 최단 경로를 **계속해서 갱신**하는 것이다.
- 노드가 많고 간선이 적을 때는 **인접 리스트**, 노드가 적고 간선이 많을 때는 **인접 행렬**이 유리하다.
- 다만, **가중치가 모두 같을 경우 BFS**로도 가능하다.(먼저 큐에 들어간 노드가 결국 거리가 짧은 노드라는 것이 항상 성립)

```python
# 1. 인접 리스트 (주로 사용)
graph = [[] for _ in range(N+1)]

# 2. 인접 행렬
graph = [[0] * (N+1) for _ in range(N+1)]
```

## 코드

```python
import heapq

graph = [[] for _ in range(num_node)]0
for _ in range(num_edge):
    s, e, w = map(int, input().split())
    graph[s].append((e, w))

def dijkstra(graph, start): 
    distances = {n: float('inf') for n in graph}
    distances[start] = 0
    pq = [(0, start)] 
    
    while pq:
        current_distance, current_node = heapq.heappop(pq)
        
        if current_distance > distances[current_node]: # 이미 최단 거리면 건너뛰기!
            continue
        
        for neighbor, weight in graph[current_node]:
            distance = current_distance + weight
            
            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(pq, (distance, neighbor))
    
    return distances
```

1. 필요한 것들을 초기화 한다.

```
distances = {0: inf, 1: inf, 2: inf, 3: inf} // inf는 무한대를 뜻함
distances[0] = 0

distances = {0: 0, 1: inf, 2: inf, 3: inf} // 0은 자기자신을 가리키기에 0임
pq = [(0, 0)]   # (거리, 노드)
```

1. 다음 거리 딕셔너리를 만든다. 다만, 아직 거리를 모르기에 무한대를 넣어둔다.
2. 이후, 처음 시작은 0으로 시작한다. 
3. heapq는 파이썬에서 기본 **min_heap**이기에 최소값을 보장한다.
4. 현재 경로와 현재 노드를 파악한 후, 현재 경로가 distances의 현재 거리보다 크면 이미 최소 경로를 찾았기에 continue로 통과한다.
5. 이후, 이웃과 거리들을 반복문으로 돌리면서 최소 경로보다 작은 경로가 있는지 확인하고 다시 이웃을 heap에 넣는다.
